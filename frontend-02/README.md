# Frontend-02 - React Blog Application

Simple React blog application with authentication and CRUD operations.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- **Home Page**: Displays last 3 active blogs
- **Blog Listing**: View all active blogs
- **Blog Detail**: View individual blog posts
- **Create/Edit**: Create and edit blog posts (requires authentication)
- **Authentication**: Login/Register functionality with JWT tokens stored in cookies

## Tech Stack

- React 18
- React Router 6
- Vite
- Vanilla CSS

## Backend

Make sure the backend API is running at `http://127.0.0.1:5000/api`
